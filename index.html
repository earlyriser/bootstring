How it works

Babelwhale is completely flexible, it's simple a collection of json files mapping the basic words and phrases an internationalized app needs. Just download it from Github and adapt it as you need.

A language file looks like this:
{
"hello":"Hola",
"profile":"Perfil"
}

Benefits

If you internationalize your app from the start you're getting: - Better code. - More users - Flexible UI
